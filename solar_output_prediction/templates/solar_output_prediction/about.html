The solar output calculator

This is a tool I built as a project for the Data Science Immersive course at General Assembly. The prediction algorithm is based on data collected from the National Renewable Energy Laboratory. This came both from their OpenPV Project, which collects data connected to over one million individual solar installations, and from their National Solar Radiation Database (NSRDB) API, which provides irradiance, weather, and windspeed data. For more on the machine learning algorithms used to build this calculator, check out my github (link below). Thanks for checking this out!
